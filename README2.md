
# How to clone this repo with the submodules
```bash
git clone --recurse-submodules --shallow-submodules --config submodule.src/robomaster_ros.url=https://github.com/EliaCereda/robomaster_ros.git --config submodule.src/robomaster_sim.url=https://github.com/jeguzzi/robomaster_sim.git --config submodule.src/simExtROS2.url=https://github.com/EliaCereda/simExtROS2.git https://github.com/USI-Projects-Collection/robotics_assignment2.git
```
